# Mockingbird
![CI](https://github.com/lekhacman/mockingbird/actions/workflows/ci.yml/badge.svg)

Mock service for developers by developers.

![Mockingbird](https://duckduckgo.com/i/11332c49.jpg)

## Cheat sheet
- path variables: 
  - `/resources/:id`
  - `ctx.request.params`
- request params: 
  - `/resources?txt=hello`
  - `ctx.request.query`
- request body: `ctx.request.body`
